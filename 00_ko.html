안녕하세요,

Tour of Rust로의 여행에 오신 것을 환영합니다. Rust 프로그래밍 언어의 특징을 하나씩 알아보려 합니다. Rust는 배우기 어려운 언어로 인식되고는 합니다. 그러나, 복잡한 내용을 다루기 전에 알아봐야 할 많은 것들이 있다는 것을 알 수 있기를 희망합니다.

이 가이드는 다른 언어로도 존재합니다:

  • 독일어
  • 영어
  • 스페인어
  • 프랑스어
  • Interlingue
  • Magyar
  • 브라질 포르투갈어
  • 러시아어
  • 중국어
  • 일본어
  • 한국어

내용에 대해 제안이 있거나 번역에 기여하고 싶은 경우, Rust로의 여행 저장소를 확인해주세요 github repository.
